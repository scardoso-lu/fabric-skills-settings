# Security Review

## Scope

- Environment: sandbox / production-bound handoff
- Pipeline or asset:
- Source:
- Target:

## Checks

- [ ] Codex did not access production.
- [ ] Target environment is explicit.
- [ ] Azure Key Vault or approved secret-store names are used.
- [ ] No secret values are present.
- [ ] Sensitive fields are classified.
- [ ] Least privilege is documented.
- [ ] No broad grants are proposed.
- [ ] RLS/OLS/view-level security is considered where needed.
- [ ] Sandbox validation evidence is available.
- [ ] Human production handoff exists if production-bound.
